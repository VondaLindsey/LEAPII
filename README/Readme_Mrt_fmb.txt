Object Name : Mrt.fmb

Description :
     There has been no functional change to the Screen. The base mrt form( Mass RTW and returns) was modified to default the Inventory status field . The USer can now only select the Inventory Status as Available whil creating a Mass transfer. The screen was also modified to display the custom RTV reason codes as displayed in the Mass RTV form. 


Algorithm : 
     - The inventory Status Field in the Mass transfer form is defaulted to "Available" when the user creates a Mass Transfer.
     - The Custom RTV reason codes are added to the Reason Code LOV for the returns.
